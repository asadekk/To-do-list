# 📝 JavaScript Todo App

A simple Todo App built with **HTML**, **CSS**, and **JavaScript**. Tasks are saved in **localStorage**, so they remain available after refreshing the page.

## 🌐 Live Demo

🔗https://todolist-two-iota-58.vercel.app

## ✨ Features

- Add new tasks
- Delete tasks
- Save tasks with localStorage
- Load saved tasks on page refresh

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
